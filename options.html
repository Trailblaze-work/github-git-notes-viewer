GitHub Git Notes Viewer

GitHub Personal Access Token

Only needed for private repos. Public repos work without a token.
Create a token with the repo scope (full access to private repositories).

Additional Note Refs

Extra refs to check, one per line. The extension already auto-discovers refs and always checks refs/notes/commits and refs/notes/claude-prompt-trail.

Cache

Notes trees are cached in memory for 5 minutes to reduce API calls.
